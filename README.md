# student_manager
<학생 정보 관리 프로그램>

프로그램 설명: GUI 기반으로 학생 정보(이름, 학번, 학과)를 추가, 삭제, 저장, 불러오기 기능을 제공하는 Python 프로그램

프로그램 화면 구성:
1. 학생 정보 입력 영역- 이름 / 학번 / 학과 입력 후 “추가” 버튼 클릭
2. 학생 목록 영역- 추가된 학생들이 테이블 형태로 표시됨
3. 기능 버튼 영역- 삭제, 파일로 저장 (JSON / CSV / XLSX), 파일 불러오기

지원 포맷: JSON (*.json), CSV (*.csv), Excel (*.xlsx)
